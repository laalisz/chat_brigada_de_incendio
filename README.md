## Sobre o Projeto
Este projeto é uma solução digital interativa focada em automação de atendimento e segurança patrimonial. Trata-se de um chatbot desenvolvido para atuar como um Assistente Técnico Especialista em diretrizes da Brigada de Incêndio. O sistema tem como objetivo fornecer respostas rápidas, precisas e estruturadas sobre protocolos de emergência, evacuação e primeiros socorros.

## Funcionalidades Principais

* **Processamento de Linguagem Natural Local:** O sistema utiliza um motor de busca interno para identificar a intenção do usuário.
* **Base de Conhecimento Modular:** Arquitetura de dados separada da interface.
* **Navegação Guiada:** Geração dinâmica de botões de ação rápida.
* **Feedback Visual e UX:** Animações de digitação e bloqueio temporal de formulário.
* **Acessibilidade:** Suporte a leitores de tela integrado através de atributos ARIA.

## Como Executar o Projeto

O projeto utiliza apenas JavaScript Vanilla (puro). Nenhuma instalação via NPM é necessária.

```bash
# 1. Faça o clone do repositório
git clone [https://github.com/laalisz/chat_brigada_de_incendio.git](https://github.com/laalisz/chat_brigada_de_incendio.git)

# 2. Navegue até a pasta
cd chat_brigada_de_incendio

# 3. Abra o arquivo no navegador
start index.html
